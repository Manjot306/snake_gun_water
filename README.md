# 🔱 Gun Water Snake

A modern, premium web-based implementation of the classic **Gun Water Snake** game built using **HTML, CSS, and JavaScript**.

🌐 Live Demo: https://classy-pony-543bd0.netlify.app/

---

## 🎮 About The Game

Gun Water Snake is a fun strategy game where you play against the computer.

### Rules

- 🔫 Gun defeats 🐍 Snake
- 🐍 Snake defeats 💧 Water
- 💧 Water defeats 🔫 Gun
- 🤝 Same choice results in a Draw

The computer randomly selects a choice and the winner is determined according to the rules above. Similar to Rock-Paper-Scissors, every option has one strength and one weakness. :contentReference[oaicite:0]{index=0}
